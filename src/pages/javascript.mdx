### Variable declarations

#### const vs let

Use `const` or `let` instead of `var`. `const` means that the identifier can't be reassigned. `let` should be used for reassignable variables. Note, `const` and `let` are both scoped to blocks.

```js
const interval = 2

let i = 0
i = i + 1
```
